# Личный кабинет заочного студента (MVP)

Простой, но качественно спроектированный кабинет студента для:
- отправки работ преподавателю,
- просмотра оценок и зачётов,
- доступа к материалам и расписанию.

## Технологии
- Python (FastAPI)
- SQLite (реляционная БД)
- HTML/Bootstrap (адаптивный фронтенд)

## Цель
Создать MVP с фокусом на:
- качественную архитектуру данных,
- безопасное хранение файлов,
- понятный и расширяемый код.
